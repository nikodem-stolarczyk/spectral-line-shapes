Configuration of the weideman64_dll.vi file:

Copy the weideman64_dll.vi and erf.dll files to your computer.
Open weideman64_dll.vi block diagram and "Call Library Function" block.
Link the current location of the erf.dll in the "Call Library Function" block under "Library Name or Path".
Choose "Function Name" as __auxiliary_mod_MOD_cef.
